# Weather API App

> Built using React App for COMP3123 
> Using OpenWeather API to find weather for all cities and locations
> Use vercel link and input weather
> https://101486489-comp3123-labtest2.vercel.app/

## Features

> Search for any city 
> Displays temp, weather condition, weather icon, humidity, pressure.
